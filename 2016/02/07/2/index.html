Python复习笔记2--语句和文件

发表于 2016-02-07   |  

1、语句

赋值

对换a,b的值
传统写法

a=9,b=5
temp=a
a=b
b=temp

Python写法：

>>>a=3
>>>b=5
>>>a,b=b,a
>>>a
5
>>>b
3

因为Python中的变量是指向对象的，所以a,b=b,a相当于将a,b标签对调，即让a指向b所指向的对象，让b指向a所指向的对象

2、条件语句

三元操作符

>>>a=8
>>>b=9 if (a==8) else 10
>>>b
9
>>>b=9 if (a==9) else 10
>>>b
10

A if X else B
如果X为真就执行A否则执行B

3、循环语句

for循环

所有序列类型的对象都可以用for来循环

range函数

一般形式是range(start, stop[, step])

  • start：开始数值，默认为0,也就是如果不写这项，就是认为start=0
  • stop：结束的数值，必须要写的。
  • step：变化的步长，默认是1,也就是不写，就是认为步长为1。坚决不能为0

取字典里的键值对

>>>adict={'amy':8,'bob':6,'cindy':3}
>>>adict
{'amy':8,'bob':6,'cindy':3}
>>>for k,v in adict.iteritems():
       print k,v
amy 8
bob 6
cindy 3
>>> for k in adict.keys():
       print k, adict[k]
amy 8
bob 6
cindy 3

使用迭代器的效率会更高，一般不推荐使用第二种方法，对单独取keys或单独取values同理，推荐使用迭代器

zip函数

>>>a='hello'
>>>b='wherever'
>>>zip(a,b)
[('h','w'),('e','h'),('l',e'),('l','r'),('o','e')]
>>>adict
{'amy':8,'bob':6,'cindy':3}
>>>bdict={'david':4,'eric'=5}
>>>bdict
{'eric':5,'david':4}
>>>zip(adict,bdict)
[('amy','eric'),('bob','david')]

zip是一个内置函数，它的参数必须是某种序列数据类型，如果是字典，那么键视为序列。然后将序列对应的元素依次组成元组，做为一个list的元素。如果序列长度不同，那么就以最短序列的长度为准。

例1： 解决两个列表中的元素相加的问题

>>>a=[1,3,2,4,5]
>>>b=[2,1,3,5,3]
>>>for k,v in zip(a,b):
       print k+v,
3,4,5,9,8

例2： 把字典中的键变成值，值变成键

>>>adict
{'amy':8,'bob':6,'cindy':3}
>>>dict(zip(adict.values(),adict.keys())))
{8:'amy',6:'bob',3:'cindy'}

enumerate

enumerate是枚举类型的对象，带索引值，如：

>>>for k,v in enumerate(adict,start=1):
       print k,v
1 amy
2 bob
3 cindy

enumerate对象的内容可以用list进行转换，也就是说是可以迭代的

list解析

去掉每个元素前后的逗号

>>>squares=[',111','23,43','354,']
>>>[i.strip(',') for i in squares]
['111','23,43','354']

while语句

while…less循环
while的循环条件不满足后就执行less中的语句

>>>a=0
>>>while a<3:
       print a,
       a +=1
  else: print a, "is not less than 3"
0 1 2 
3 is not less than 3

for…else循环
同样else后的语句也是在for所有循环后做

4、文件

打开读取文件

>>>f=open('130.txt')
>>>for line in f:
       print line
>>>f.close()

创建写入文件

>>>f=open('131.txt','w')
>>>f.write('hello, world.')
>>>f.close()

使用with

>>>with open('131.txt','a') as f:
       f.write("\n This is about with\n")
...
>>>with open('131.txt','r') as f:
       print f.read()

其中a是追加模式，即一打开文件光标就移到文件尾端，如果不存在这个文件则会创建文件。
使用with方法就不用close()了，可以执行完操作自动关闭文件。

查看文件状态

>>>import os
>>>file_stat=os.stat('baiducookie.txt')
>>>file_stat #可以查看文件状态
>>>import time
>>>time.localtime(file_stat.st_ctime)#可以查看文件创建时间

read/readline/readlines

read方法，如果指定了参数就读这么长的文件，如果没指定参数就读取全文，把读出来的内容全部放到一个字符串中。
readline()方法，执行一次读一行，可迭代。
readlines()方法，先执行readline()方法，将文件中的每行变成一个列表的元素存储在一个列表中，然后返回列表，也是可迭代的

f=open('131.txt','r')
while True:
    if not line:
        break
    print line,
#可以打印整个文件，有分行
f=open('131.txt')
content=f.readlines()
content#会返回一个列表里面装着131.txt的每一行作为一个元素
for line in content:
    print line,
#可以打印整个文件，有分行

fileinput模块

可以用来打开比较大的文件，可迭代

>>>import fileinput
>>>for line in fileinput.input('131.txt'):
       print line,
       #可以打印整个文件，有分行

练习

1、按照下面的要求实现对列表的操作：

产生一个列表，其中有40个元素，每个元素是0到100的一个随机整数
如果这个列表中的数据代表着某个班级40人的分数，请计算成绩低于平均分的学生人数，并输出
对上面的列表元素从大到小排序

import random
l = []
for i in range(40):
    l.append(random.randint(1,100))

print l #产生列表，每个元素是0到100间的一个随机整数

ad = 0
for i in l:
    ad += i  #求列表中每个元素相加的和
print ad/40.0  #求平均分
count = 0
for i in l:
    if i<ad/40:
        count += 1
print count   #求低于平均分的人数

a = sorted(l,reverse=True)
print a  #排序

改进：
形成列表可以用更简单的方式，直接list化
score = [random.randint(0,100) for i in range(40)]
列表求和：用sum函数
sum_score = sum(score)
找小于平均数的并形成新列表：用list解析
less_ave = len([i for i in score if i<ave_num])
